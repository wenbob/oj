# 模拟题 1：下课时间

## 难度

入门

## 分类

日期与时间换算

## 题目描述

小杨在某一时刻开始上一节课，这节课持续若干分钟。已知开始上课的时和分，请你计算下课时刻的时和分。

保证下课时刻仍在当天。

## 输入格式

输入三行。

第一行包含一个整数 `h`，表示开始时刻的小时。

第二行包含一个整数 `m`，表示开始时刻的分钟。

第三行包含一个整数 `t`，表示课程持续的分钟数。

## 输出格式

输出一行，包含两个整数，分别表示下课时刻的时和分，中间用一个空格隔开。

## 样例

### 输入样例 1

```text
8
20
45
```

### 输出样例 1

```text
9 5
```

### 输入样例 2

```text
14
0
90
```

### 输出样例 2

```text
15 30
```

## 数据范围

对于所有测试点，保证 `0 <= h <= 23`，`0 <= m <= 59`，`1 <= t <= 300`，且下课时刻仍在当天。

## 题目描述

小杨的目标分数是 `a`。现在有两个练习成绩 `b` 和 `c`，请你输出更接近目标分数的那个成绩。如果两个成绩与目标分数的差一样大，输出较小的成绩。

## 输入格式

输入三行。

第一行包含一个整数 `a`，表示目标分数。

第二行包含一个整数 `b`。

第三行包含一个整数 `c`。

## 输出格式

输出一行，包含一个整数，表示更接近目标分数的成绩。

## 样例

### 输入样例 1

```text
80
75
85
```

### 输出样例 1

```text
75
```

### 输入样例 2

```text
60
70
50
```

### 输出样例 2

```text
50
```

## 数据范围

对于所有测试点，保证 `0 <= a,b,c <= 100`。

```text
56
90
78
```

### 输出样例 1

```text
90 56
```

### 输入样例 2

```text
100
0
60
```

### 输出样例 2

```text
100 0
```

对于所有测试点，保证三个成绩均为 `0` 到 `100` 之间的整数。

# 模拟题 3：最合适的座位

## 难度

入门

## 分类

最值比较与绝对值

## 题目描述

小杨想选择一个离讲台距离最合适的座位。讲台到小杨当前位置的理想距离是 `x`，现在有四个空座位，它们到讲台的距离分别为 `a`、`b`、`c`、`d`。

请输出与 `x` 最接近的座位距离。如果有多个座位同样接近，输出距离较小的那个。

## 输入格式

输入五行。

第一行包含一个整数 `x`，表示理想距离。

接下来四行分别包含整数 `a`、`b`、`c`、`d`，表示四个空座位到讲台的距离。

## 输出格式

输出一行，包含一个整数，表示最合适的座位距离。

## 样例

### 输入样例 1

```text
150
165
135
133
160
```

### 输出样例 1

```text
160
```

### 输入样例 2

```text
100
110
90
95
130
```

### 输出样例 2

```text
95
```

## 数据范围

对于所有测试点，保证 `1 <= x,a,b,c,d <= 300`，且四个空座位的距离互不相同。
