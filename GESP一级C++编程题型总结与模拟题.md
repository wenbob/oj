## 二、题型归纳

| 专题 | 覆盖真题 | 常见考点 | 常见陷阱 |
|---|---|---|---|
| 日期与时间换算 | 每月天数、时间规划、小杨的考试、休息时间 | 取余、进位、小时分钟秒转换、闰年判断 | 星期 7 或星期日的表示、秒转时分秒的顺序 |
| 购物计费与整除取余 | 买文具、小杨买书、小杨购物、图书馆里的老鼠、假期阅读 | 总价、预算、整除、余数、向上取整、最小值 | 是否需要买完整件、是否发生部分消耗 |
| 分段判断与格式化实数 | 温度转换、商店折扣、爱心快递、手机电量显示 | if/else、浮点数计算、保留小数、取较小值 | 输出文字必须完全一致，小数位数必须固定 |
| 循环枚举与条件计数 | 小杨报数、幸运数、美丽数字、奇数和偶数、四舍五入 | for/while、多组输入、倍数判断、计数或求和 | 同时满足或排除条件时容易用错 `&&` 和 `||` |
| 因数倍数与特殊数判定 | 长方形面积、找因数、立方数、值日 | 因数、公倍数、完全平方或立方 | 从 1 开始枚举，循环边界要包含目标数 |
| 累加与图形数量 | 累计相加、金字塔 | 等差累加、平方和、嵌套意义的累加 | 题意是“每层”还是“累计到每层” |
| 数位处理 | 数字替换，以及选择题中的数位和、奇数位判断 | `%10` 取个位、`/10` 去个位、按位重组 | 重组时要用位权 `1,10,100...` 保持原顺序 |
| 最值比较与绝对值 | 交朋友 | 两数或多数比较、绝对值、平局规则 | 距离相等时要按题目要求选较小或较大 |

## 三、专题模拟真题

## 专题一：日期与时间换算

### 模拟题 1：下课时间

#### 描述

小杨在某一时刻开始上一节课，这节课持续若干分钟。已知开始上课的时和分，请你计算下课时刻的时和分。

保证下课时刻仍在当天。

#### 输入

输入三行。

第一行包含一个整数 `h`，表示开始时刻的小时。

第二行包含一个整数 `m`，表示开始时刻的分钟。

第三行包含一个整数 `t`，表示课程持续的分钟数。

#### 输出

输出一行，包含两个整数，分别表示下课时刻的时和分，中间用一个空格隔开。

#### 样例

输入：

```text
8
20
45
```

输出：

```text
9 5
```

#### 数据范围

对于所有测试点，保证 `0 <= h <= 23`，`0 <= m <= 59`，`1 <= t <= 300`，且下课时刻仍在当天。

### 模拟题 2：训练日

#### 描述

今天是星期 `w`，其中 `1` 到 `7` 分别表示星期一到星期日。小杨还要过 `d` 天参加训练，请你输出训练当天是星期几。

#### 输入

输入两行。

第一行包含一个整数 `w`，表示今天是星期几。

第二行包含一个整数 `d`，表示还要过多少天参加训练。

#### 输出

输出一个整数，表示训练当天是星期几。

#### 样例

输入：

```text
5
3
```

输出：

```text
1
```

#### 数据范围

对于所有测试点，保证 `1 <= w <= 7`，`0 <= d <= 10000`。

### 模拟题 3：闹钟提醒

#### 描述

小杨设置了一个闹钟。现在给出当前时刻的时、分、秒，以及闹钟会在 `t` 秒后响起。请你输出闹钟响起时是第几天，以及当天的时、分、秒。

当前这一天记为第 `1` 天。

#### 输入

输入四行。

前三行分别包含整数 `h`、`m`、`s`，表示当前时刻的时、分、秒。

第四行包含一个整数 `t`，表示闹钟会在多少秒后响起。

#### 输出

输出一行，包含四个整数，分别表示第几天、时、分、秒，相邻两个整数之间用一个空格隔开。

#### 样例

输入：

```text
23
59
50
15
```

输出：

```text
2 0 0 5
```

#### 数据范围

对于所有测试点，保证 `0 <= h <= 23`，`0 <= m <= 59`，`0 <= s <= 59`，`0 <= t <= 200000`。
