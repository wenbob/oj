### 第 4 题

### 第 5 题

### 第 6 题

下面程序用于判断整数 `n` 是否既能被 2 整除，又能被 3 整除，横线处应填入（ ）。

```cpp

    cout << "Yes";
else

```

A. `n % 2 == 0 || n % 3 == 0`

B. `n % 2 == 0 && n % 3 == 0`

C. `n / 2 == 0 && n / 3 == 0`

D. `n % 2 == 1 && n % 3 == 1`

## 二、判断题（每题 2 分，共 20 分）

### 第 1 题

C++ 表达式 `int(3.14)` 的值为 `3`。（ ）

### 第 3 题

### 第 4 题

表达式 `8 / 3` 和 `8 % 3` 的值相同。（ ）

### 第 5 题

执行语句 `cout << "Hello";` 会输出 `Hello`。（ ）

### 第 6 题

任何一个 `while` 循环都一定会执行至少一次循环体。（ ）

### 第 7 题

`continue` 语句可以跳过本次循环中它后面的语句，直接进入下一次循环判断。（ ）

### 第 8 题

在 C++ 中，字符常量 `'1'` 和整数 `1` 的值完全相同。（ ）

### 第 9 题

若 `int a = 6;`，则表达式 `a % 2 == 0` 的值为真。（ ）

### 第 10 题

## 三、答案与简要解析

### 单选题答案

| 题号 | 答案 | 简要解析 |
|---|---|---|
| 1 | C | `int` 是 C++ 关键字，不能作为变量名。 |
| 2 | C | 先算乘法和取余：`3*2=6`，`6%5=1`，所以 `12-1=11`。 |
| 3 | B | `17/5=3`，`17%5=2`，和为 `5`。 |
| 4 | B | `7%3` 计算结果是 `1`，`"7%3"` 是字符串原样输出。 |
| 5 | C | `'A'+2` 对应字符 `'C'`。 |
| 6 | B | “既能被 2 整除，又能被 3 整除”要用 `&&`。 |
| 7 | B | `i` 依次为 `1,3,5`。 |
| 8 | B | 每次输出当前 `x`，再减 1，所以输出 `321`。 |
| 9 | A | 星期 5 过 3 天是星期 1。 |
| 10 | C | 上午 10 点后 20 小时是第二天 6 点。 |
| 11 | C | `a>3` 成立，输出 `a+1=6`。 |
| 12 | A | 前半部分为真，后半部分为假，真且假为假，即 `0`。 |
| 13 | B | `9/2=4`，`9%2=1`。 |
| 14 | A | Windows 是操作系统。 |
| 15 | C | `1+2+3+4=10`。 |

### 判断题答案

| 题号 | 答案 | 简要解析 |
|---|---|---|
| 1 | 对 | `int(3.14)` 会取整数部分，结果是 `3`。 |
| 2 | 错 | 变量名不能以数字开头。 |
| 3 | 对 | C++ 中整数也可以作为条件，0 为假，非 0 为真。 |
| 4 | 对 | `8/3=2`，`8%3=2`，这两个表达式在本题中的值相同。 |
| 5 | 对 | 双引号中的内容会原样输出。 |
| 6 | 错 | `while` 是先判断再执行，可能一次也不执行。 |
| 7 | 对 | `continue` 会跳过本次循环后面的语句。 |
| 8 | 错 | `'1'` 是字符，ASCII 码为 49，不等于整数 1。 |
| 9 | 对 | `6` 能被 `2` 整除。 |
| 10 | 错 | Dev-C++ 是编程开发工具，不是操作系统。 |
