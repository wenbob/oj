## 专题一：区间枚举与素数判断

### 模拟题 1：区间素数

#### 描述

给定两个正整数 `L` 和 `R`，请你统计 `L` 到 `R` 之间有多少个素数。范围包含 `L` 和 `R`。

一个大于 `1` 的正整数，如果除了 `1` 和它本身以外，不能被其他正整数整除，则称为素数。

#### 输入

输入两行。

第一行包含一个整数 `L`。

第二行包含一个整数 `R`。

#### 输出

输出一行，包含一个整数，表示区间中的素数个数。

#### 样例

输入：

```text
2
10
```

输出：

```text
4
```

#### 数据范围

对于所有测试点，保证 `2 <= L <= R <= 5000`。

#### 输出

#### 样例

输入：

```text

```

输出：

```text
17
```

#### 数据范围

对于所有测试点，保证 `2 <= n <= 10000`。


### 模拟题 2：整数面积三角形

#### 描述

直角三角形的两条直角边长分别为正整数 `a` 和 `b`，面积为 `a*b/2`。

给定 `n`，请你统计有多少对正整数 `a` 和 `b`，满足 `1 <= a <= b <= n`，且对应直角三角形面积为整数。

#### 输入

输入一行，包含一个整数 `n`。

#### 输出

输出一行，包含一个整数，表示符合条件的三角形数量。

#### 样例

输入：

```text
3
```

输出：

```text
3
```

#### 数据范围

对于所有测试点，保证 `1 <= n <= 1000`。

### 模拟题 3：圆内格点

#### 描述

给定一个 `H` 行 `W` 列的网格，行号从 `1` 到 `H`，列号从 `1` 到 `W`。

如果格子 `(i,j)` 满足 `i*i + j*j <= R*R`，则称这个格子在圆内。请你统计圆内格子的数量。

#### 输入

输入三行，分别包含整数 `H`、`W`、`R`。

#### 输出

输出一行，包含一个整数，表示圆内格子的数量。

#### 样例

输入：

```text
3
4
3
```

输出：

```text
4
```

#### 数据范围

对于所有测试点，保证 `1 <= H,W,R <= 500`。
